# Deixis Logo — Variations & Usage

Source: Deixis_Logo_Variations.pdf

## Logo Lockups

- Primary (Horizontal): Gyroscope + wordmark. Used on website header, social media, email.
- Secondary (Stacked): Gyroscope above wordmark. Used on mobile and small formats.
- Mark Only: Gyroscope alone. Used on favicon, app icon, avatar.

## Color Variations

- Full color: Light backgrounds
- White/reversed: Dark backgrounds
- Monochrome black: Print materials
- Monochrome gray: Secondary applications

## Clear Space

Minimum equal to gyroscope height on all sides. Never crowd the logo.

## Minimum Sizes

- Digital: 32px height (horizontal), 24px (mark only)
- Print: 0.5in width (horizontal), 0.25in (mark only)

## Prohibited Uses

- No recolor outside specified variations
- No rotation, skew, or distortion
- No drop shadows or effects
- No low-contrast backgrounds
- No separation of mark from wordmark in primary lockup

## Formats Needed

SVG (web primary), PNG transparent (web secondary), PNG white bg (email), PDF vector (print), .ico (favicon)
