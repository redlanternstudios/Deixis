# Deixis Digital Gallery

A digital gallery platform — visual-first, curated, context-aware.

Built with Next.js 16, React 19, TypeScript 5, and Tailwind CSS v4.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Key Documents

- [DEIXIS_BUILD_SPEC.md](./DEIXIS_BUILD_SPEC.md) — Build phases, architecture, deployment targets
- [Preliminary_Deixis.md](./Preliminary_Deixis.md) — Brand reference (colors, voice, positioning)
- [Deixis_Logo_Variations.md](./Deixis_Logo_Variations.md) — Logo concept catalog and asset requirements

## Deploy

```bash
npm run build
npm run start
```

Deploy target: Vercel.
